HTML - язык гипертекстовой разметки

Файлы .html открываются в браузере

Справочник тега

  • Теги для текста
  • Теги для выделдения текста
  • Теги для графических элементов
  • Теги задающие структуру страницы

1.Теги для текста
  • Тег p от английского paragraph абзац
  • h1..h6 от англ. heading заголовки
  • ul/ol Нумерованный и маркерованный списки
  • li от англ. list item - элемент списка
  • br - перенос строки
  • a от англ. anchor - ссылка

2. Теги для выделения текста

  • b - жирный
  • i - курсивный
  • del - перечеркнутый

3. Теги для графических элементов

  • hr -
    линия от horizontal rul
  • hr color - устанавливает линии определенный цвет
  • hr size - устанавливает высоту линии

4. Теги задающие структуру страницы

  • !DOCTYPE html - не тег, а инсрукция
  • html - заключает в себе содержащее веб-страницы
  • head - для хранения служебной информации о странице
  • body - содержимое документа Html
  • section - задает раздел документа
  • div - универсальный контейнер
